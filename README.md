# Stock-Web

Sistema web simples de gerenciamento de estoque, desenvolvido em Python com Flask, HTML e JSON.

## Funcionalidades
- Adicionar, remover e atualizar produtos
- Buscar produtos por nome
- Persistência dos dados em arquivo JSON

- ## Tecnologias
- Python 3
- Flask
- HTML5 e CSS3
- JSON

  ## Como executar
1. Clone o repositório:  
   `git clone https://github.com/GiovaneMiq/Stock-Web.git`
2. Instale o Flask:  
   `pip install flask`
3. Execute o servidor:  
   `python app.py`
4. Acesse no navegador:  
   `http://127.0.0.1:5000`
